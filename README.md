# Neurosim — Інтерактивний симулятор нейронних мереж

## Опис
Веб-симулятор для візуальної демонстрації роботи штучного нейрона та багатошарової нейронної мережі.

## Функціонал
- Моделювання одного нейрона з різними функціями активації
- Побудова MLP довільної архітектури
- Навчання на логічних задачах (AND, OR, XOR)
- Порівняння функцій активації та їх похідних
- Експорт результатів у JSON, CSV, PNG

## Запуск
1. Завантажити файл `index.html`
2. Відкрити у будь-якому сучасному браузері

## Технології
- HTML5, CSS3, JavaScript (ES6+)
- Canvas API
